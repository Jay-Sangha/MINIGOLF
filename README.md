# Mini Golf

Browser mini golf game built with Three.js.

## Play

Open [jay-sangha.github.io/MINIGOLF](https://jay-sangha.github.io/MINIGOLF/) or run locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Controls

- Move the mouse to aim, click the green to lock aim
- Adjust power and click **Putt!**
- **WASD** to rotate the camera, right-drag to orbit, scroll to zoom
- **R** to restart after a water hazard
